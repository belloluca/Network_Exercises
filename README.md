# Computer Networks Laboratory - C++ Exercises

Collection of C++ exercises developed during the **Computer Networks Laboratory** course.

The repository contains practical implementations of the main networking concepts studied during the course, including socket programming, client-server architectures, multithreading, reliable and unreliable communications, and distributed systems.

## 📚 Topics Covered

- TCP Client/Server applications
- UDP Client/Server applications
- Multithreaded servers
- Socket programming
- Reliable and unreliable communication
- Network protocols
- Client management
- Message parsing
- Synchronization using mutexes
- File logging
- Distributed system simulations

## 🚀 Exercises

The repository includes implementations of exercises such as:

- 📡 Sensor Network (Sensor Node, Central Node and Control Node)
- ✂️ Rock, Paper, Scissors multiplayer game
- 💬 Peer-to-Peer Chat
- 🎮 Multiplayer networking exercises
- 🖥️ TCP and UDP communication examples
- 🔄 Multithreading exercises
- 📂 Client/server architecture examples

Additional exercises may be added over time.

## 🛠️ Technologies

- C++
- Threads
- TCP Sockets
- UDP Sockets
- Linux Networking APIs
- STL (map, vector, queue, mutex, string)

## 🎯 Learning Objectives

The purpose of this repository is to practice and better understand:

- Network programming in C++
- TCP and UDP communication
- Concurrent programming
- Thread synchronization
- Client-server architectures
- Peer-to-peer communication
- Design of distributed systems
- Network protocol implementation

## ▶️ Requirements

- Linux
- g++
- POSIX socket support
- C++17 (or newer)

Example compilation:

```bash
g++ server.cpp -o server -pthread
g++ client.cpp -o client -pthread
```

Run:

```bash
./server
./client
```

These exercises were developed for educational purposes as part of the Computer Networks Laboratory course. The implementations focus on understanding networking concepts and are intended as learning examples.
